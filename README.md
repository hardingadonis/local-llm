# Local LLM
A local LLM chatbot

## Requirements
- Python 3
- `virtualenv` package

## Installation
- Clone repository
```bash
git clone https://github.com/hardingadonis/local-llm.git
cd local-llm/
```

- Create virtual environment
```bash
virtualenv .venv
```

- Activate virtual environment
```bash
"./.venv/Scripts/activate"
```

- Install dependencies
```bash
pip install -r requirements.txt
```

- Run application
```bash
python main.py
```
